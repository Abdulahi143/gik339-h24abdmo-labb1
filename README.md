GIK339 – Labb 1: JavaScript intro, variabler och funktioner

Detta projekt innehåller lösningar till Labb 1 i kursen GIK339 – JavaScript intro, variabler och funktioner.
Syftet med labben är att träna på:

Grundläggande HTML och koppling till en extern JavaScript-fil

Skillnader mellan var, let och const

Jämförelser med == och === samt hantering av specialvärden som NaN, null och undefined

Funktioner, parametrar och argument

Användning av ternary operator (?:)

Innehåll

index.html – grundläggande HTML-struktur med koppling till script.js

script.js – innehåller alla uppgifter, reflektioner och funktioner från labben

Reflektion uppgift 2: variabler och scope

Reflektion uppgift 3: jämförelser och specialvärden

Reflektion uppgift 4: funktioner

Funktion multSum() som beräknar summa och produkt av två tal (med ternary operator)
